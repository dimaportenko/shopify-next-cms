
## Mission Answer

Free template devs clone via CLI (create-shopify-next-cms) to bootstrap   custom Shopify storefronts with a Puck-based CMS. The idea to eliminate some common problems for existing solutions:
- use headless app over Shopify liquid template to have freedom to build custom solution;
- have CMS to give content team ability do marketing experements;
- escape bounds of 3d party cms integrations, which force you use their data/content structure;
- seamless connection between cms and shopify data via APIs;
- simplicity of setup where all data stored in the Shopify, no extra database or other services needed.
