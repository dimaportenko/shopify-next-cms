The theme customizer is a side-panel UI that lets you tweak your shadcn/ui theme in real time. Changes are applied instantly via CSS variables and persisted to `localStorage`.

## Features

| Tab            | What it controls                                                                                                                                                    |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Colors**     | 11 semantic color groups (primary, secondary, accent, base, card, popover, muted, destructive, border & input, chart, sidebar) with search and collapsible sections |
| **Typography** | Font family (sans, serif, mono) loaded live from Google Fonts, plus letter spacing                                                                                  |
| **Other**      | HSL bulk adjustments with presets, border radius, spacing, and shadow controls (color, opacity, blur, spread, offset)                                               |

## How it works

1. Open the customizer by clicking the palette icon in the toolbar
2. Pick a color preset or tweak individual values
3. Switch between Light / Dark mode to edit each independently
4. Typography and shadow changes apply to both modes
5. Click **Copy CSS** to export the generated `--custom-property` declarations
6. Click **Reset** to restore defaults

## Architecture

- `components/theme-customizer.tsx` — main Sheet UI with Colors, Typography, Other tabs
- `hooks/use-theme-customizer.ts` — state management, CSS variable application, localStorage persistence
- `hooks/use-google-fonts.ts` — fetches font list and manages dynamic loading
- `lib/google-fonts.ts` — client-side font loading via `<link>` injection
- `lib/theme-presets.ts` — color preset definitions and type interfaces
- `app/api/fonts/route.ts` — server-side proxy for the Google Fonts API
