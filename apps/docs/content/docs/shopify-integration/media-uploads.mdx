---
title: Media Uploads
description: Uploading and browsing files via Shopify's staged upload API.
---

The CMS provides API routes for uploading images to Shopify and browsing the media library. These power the [media picker field](/docs/puck-editor/custom-fields) in the Puck editor.

## API Routes

### POST /api/cms/media/staged-upload

Prepares a file for upload by requesting a staged upload URL from Shopify.

**Request body:**

```json
{
  "filename": "photo.jpg",
  "mimeType": "image/jpeg",
  "fileSize": "204800"
}
```

**Response:**

```json
{
  "url": "https://shopify-staged-uploads.storage.googleapis.com/...",
  "resourceUrl": "https://shopify-staged-uploads.storage.googleapis.com/...",
  "parameters": [
    { "name": "Content-Type", "value": "image/jpeg" },
    { "name": "key", "value": "tmp/..." }
  ]
}
```

The client then uploads the file directly to the returned `url` as a multipart `FormData`, appending all `parameters` before the file.

### POST /api/cms/media

Registers an uploaded file in Shopify's file system via the `fileCreate` mutation.

**Request body:**

```json
{
  "resourceUrl": "https://shopify-staged-uploads.storage.googleapis.com/...",
  "filename": "photo.jpg",
  "alt": "Product photo"
}
```

**Response:**

```json
{
  "id": "gid://shopify/MediaImage/123",
  "url": "https://cdn.shopify.com/s/files/...",
  "alt": "Product photo"
}
```

If the CDN URL is not immediately available (Shopify processes files asynchronously), the route polls up to 5 times before falling back to the `resourceUrl`.

### GET /api/cms/media

Browses existing images in the Shopify media library with pagination and search.

**Query parameters:**

| Param | Type | Default | Description |
|-------|------|---------|-------------|
| `first` | number | 20 | Number of images to return |
| `after` | string | — | Cursor for pagination |
| `search` | string | — | Search by filename |

**Response:**

```json
{
  "images": [
    {
      "id": "gid://shopify/MediaImage/123",
      "url": "https://cdn.shopify.com/s/files/...",
      "alt": "Photo",
      "width": 800,
      "height": 600
    }
  ],
  "pageInfo": {
    "hasNextPage": true,
    "endCursor": "eyJsYXN0X..."
  }
}
```

## GraphQL Queries

All media-related GraphQL operations are defined in `src/lib/shopify/queries/media.admin.ts`:

| Query | Operation | Purpose |
|-------|-----------|---------|
| `STAGED_UPLOADS_CREATE` | Mutation | Get presigned upload URL |
| `FILE_CREATE` | Mutation | Register uploaded file |
| `FILE_BY_ID_QUERY` | Query | Poll for processed CDN URL |
| `FILES_QUERY` | Query | List media with pagination |

Types are generated via `pnpm codegen:admin` and imported from `@generated-types/admin.generated`.

## Image Domains

The following domains must be in `next.config.ts` `remotePatterns` for images to render:

| Domain | Purpose |
|--------|---------|
| `cdn.shopify.com` | Final CDN URLs for processed images |
| `shopify-staged-uploads.storage.googleapis.com` | Staged upload URLs (fallback) |
| `cdn.shadcnstudio.com` | Default placeholder images |
