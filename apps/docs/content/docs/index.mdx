## Introduction

Shopify Next CMS is a content management system built with Next.js and integrated with Shopify.

## Project Structure

This is a monorepo powered by [pnpm](https://pnpm.io) and [Turborepo](https://turbo.build):

```
apps/
  shopify-next-cms/   # Main Next.js application
  docs/               # Documentation (you are here)
packages/             # Shared packages
```

## Development

Start all apps in development mode:

```bash
pnpm dev
```

Build all apps:

```bash
pnpm build
```
