# TypeScript Best Practices Skill

A coding agent skill with concise TypeScript rules and review guidance for common development tasks.

## Install

```bash
npx skills add <your-github-username>/effective-typescript-skill
```

Or manually copy the folder to `.claude/skills/` in your project.

## What's Inside

- **SKILL.md** — Core rules grouped into 8 categories, plus a code review checklist
- **references/rules.md** — Complete per-item reference covering 83 practical rules

## Categories Covered

1. Type Safety Fundamentals
2. Type Design
3. Type Inference & Narrowing
4. Generics & Type-Level Programming
5. Structural Typing & Type Boundaries
6. Declarations & APIs
7. Type System Mechanics
8. Code Quality & Migration

## When It Triggers

The skill activates on TypeScript-related tasks such as writing code, reviewing code, designing types, fixing type errors, setting up `tsconfig`, working with generics, handling nulls, and migrating from JavaScript.

## Notes

This repository contains concise, task-oriented guidance intended for coding agents and developers.
